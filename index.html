Git Lapse

A optimization for git targeted visualization

Created for the Web and Mobile Research Lab, Git Lapse is a web app that enables visualization of Git commits, taking advantage of Myer's Diff Algorithm, which is optimized between texts that are similar. Myer's algorithm is ideal for capsulizing the difference in Git commits because commits rarely feature drastic changes, usually opting for incremental changes. Thus, combining Myer's Diff Algorithm with a simple text comparator enables quick generation of visualization. Learn more here.
